## 2. AWSが提供するStorageサービス

- オブジェクトストレージ（高耐久・大容量）
  - S3
    - 用途
      - バックアップ
      - データレイク
        - ビッグデータを保存する容器のこと
      - 静的コンテンツ（HTML/CSS）のホスティング
        - ※動的コンテンツ（Ruby, PHPなど）はEC2にホスティングする
    - 構成要素
      - バケット
        - 保存領域のこと。一意の名称を付ける。
      - オブジェクト
        - データそのもの。一意のURLを持つ。
      - メタデータ
        - 作成日時、サイズなど。
        - どんな情報を持たせるかはユーザがカスタマイズできる



- ファイルストレージ（スケーラブルな共有ストレージ）
  - EFS（Elastic File System）
  - FSx for Windows
  - FSx for Lustre
- ブロックストレージ（永続的なブロックストレージ）
  - EBS（Elastic Block Store）
